# Detecting Health Misinformation in COVID-19 News Headlines

## Overview
This project analyzes fake vs. real COVID-19 news headlines using Python.  
It applies sentiment analysis (TextBlob, VADER) and text visualization techniques.

## Features
- Exploratory analysis of claim length, word count, and average word length.
- Sentiment analysis using TextBlob and VADER.
- Wordclouds and bigram analysis to identify common keywords in fake vs. real headlines.
- Interactive plots using Plotly.

## Data
The dataset contains COVID-19 news headlines labeled as `Fake` or `Real`.
